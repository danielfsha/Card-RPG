# Etherion Card Database

The **Etherion** protocol utilizes a standardized card anatomy to ensure compatibility with Zero-Knowledge circuits. Each card is an instance of a **Spectral Champion** or **Artifact**.

## Card Anatomy

Every card in the deck requires the following attributes to be verifiable:

| Attribute     | Bit Range | Description                                      |
| :------------ | :-------- | :----------------------------------------------- |
| **ID**        | 0-7       | Unique identifier (0-255)                        |
| **Suit/Arch** | 8-11      | The Archetype (e.g., Valor/Sword, Harbor/Anchor) |
| **Rank**      | 12-15     | Level or Rank (1-12)                             |
| **ATK**       | 16-27     | Attack Points (0-4095)                           |
| **DEF**       | 28-39     | Defense Points (0-4095)                          |
| **Type**      | 40-43     | Monster Type (Dragon, Warrior, Machine...)       |
| **Attribute** | 44-47     | Elemental Attribute (LIGHT, DARK, FIRE...)       |
